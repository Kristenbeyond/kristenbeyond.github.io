Bootstrap Callout导致hexo博客.md文档标题无法正常显示的办法

发表于 2020-03-25 | 分类于 problem | | 次

我在写第六单元的作业时，内容写在了.md(markdown)文档里，出于测试的原因，使用了很多新添加的功能，导致了我的部分一级，二级，三级标题在渲染后无法正常显示。再删除Bootstrap Callout相关标签之后居然就恢复正常了。
