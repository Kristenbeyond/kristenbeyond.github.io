在电脑上规避学习通的一些方法

发表于 2020-03-16 | 分类于 Plugin | | 次

声明

本文章目的只是用来分享，本人的网课全部亲自听完，作业为自己做的。使用插件的原因只是因为在听网课时鼠标不能自由移动，常常有各种各样的事情找上我使我不得不（被迫）暂停视频，很是麻烦。

前期准备

1.浏览器，火狐，谷歌等等，建议谷歌内核的浏览器（国内浏览器虽然大部分都是谷歌内核，但是版本非常老旧，这里不推荐），原版谷歌浏览器当然更好的支持，但是我更喜欢新版的带谷歌内核的edge，点击下载

2.梯子，因为会使用谷歌插件，所以会用到梯子。没有的话可以使用其他非谷歌内核浏览器，如火狐浏览器。

添加Tampermonkey插件

1.进入Greasy Fork主页，添加一个脚本插件，这里推荐Tampermonkey。操作非常简单，其实进入页面方法已经告诉你了。

谷歌内核浏览器添加

进入主页，梯子打开，注意，一定要打开梯子，点击箭头指向的Chrome:Tampermonkey

接着点击安装插件，由于我已经安装了，我的显示的是从Chrome中删除。

****

安装完之后浏览器右上角有以下图标就说明安装成功了。

火狐浏览器的安装

和谷歌内核同理，链接我已经提供，点击安装即可，Firefox：Greasemonkey

学习通（超星）脚本的安装

可以在Greasy Fork的主页直接搜索超星，选择安装量最多的一个就是。

当然，如果嫌麻烦，直接点击我提供的超链接即可跳转。

然后直接点击安装。这里我已经安装了。

到此，学习通脚本已经安装好了，进入超星网站时浏览器右上角的图标有红色的数字说明已经成功，在你听课时就可以自由移动鼠标了。

插件进阶设置

自由开关

进入超星网站，点击浏览器右上角的插件图标，即可自由开关脚本。

当然，进入管理面板还可以对脚本进行删除。

安装其他脚本

其他脚本可根据需求在Greasy Fork的主页搜索，比如解除批改网复制粘贴限制，bilibili视频下载等等。大家自由探索，这里不过多说明。（保命）

脚本的进阶使用（功能介绍）

其实（学习通）脚本安装完之后就不需要手动操作了，这里算是对超星脚本的功能的说明。

除了解除鼠标限制之外还有一个功能就是自动答题。

点击一个学科的作业，进入即可自动答题，不需要操作，甚至不需要你点提交。

这里我已经做完了，只是放出一个效果图。
