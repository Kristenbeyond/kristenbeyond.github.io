git笔记(自用)

git log :查看历史提交

git status: 查看当前状态

git mist 查看提交（简易）

git reset head ~1 撤销上一次提交（仅在当次git bush中适用）

git status -s 详细（进入本地仓库的状态 ??表示未跟踪）

git remote 查看当前配置了那些远程仓库

git remote -v 详细

本地代码上传过程：

git init 初始化本地仓库

git add 添加上传文件

git commit -m “ 提交备注”

git remote add origin “url” 配置远程仓库

(git pull –rebase origin master) 合并远程仓库与本地仓库

git push -u origin master 提交至远程仓库
